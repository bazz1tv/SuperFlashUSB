HID class driver callback function for the creation of HID reports to the host.

Parameters
[in]HIDInterfaceInfoPointer to the HID class interface configuration structure being referenced
[in,out]ReportIDReport ID requested by the host if non-zero, otherwise callback should set to the generated report ID
[in]ReportTypeType of the report to create, either HID_REPORT_ITEM_In or HID_REPORT_ITEM_Feature
[out]ReportDataPointer to a buffer where the created report should be stored
[out]ReportSizeNumber of bytes written in the report (or zero if no report is to be sent)
Returns
Boolean true to force the sending of the report, false to let the library determine if it needs to be sent
